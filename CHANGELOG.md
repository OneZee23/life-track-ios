# Changelog

Все заметные изменения в проекте LifeTrack iOS.
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

---

## [0.2.5] — 2 марта 2026

### Added
- **SharedComponents.swift** — общие UI-компоненты: `NavArrowButton`, `PlaceholderView`, `StreakCardView`, `barColor`
- **computeHabitStats** в AppStore — единый метод расчёта статистики по привычкам (year/month)

### Fixed
- **DayProgressView** — показывает привычки, существовавшие на выбранную дату, а не только текущие
- **scheduleDaily()** — одно повторяющееся уведомление вместо 7 одноразовых
- **YearProgressView** — `computeYearTotals()` вызывался дважды за рендер, теперь кэшируется

### Changed
- `HabitStat` и `MonthlyStat` вынесены в Models.swift (были продублированы в 2 файлах)
- `navArrow` удалён из 6 файлов → заменён на `NavArrowButton`
- `placeholder` удалён из 3 файлов → заменён на `PlaceholderView`
- `streakCard` удалён из 2 файлов → заменён на `StreakCardView`
- `barColor` удалён из 2 файлов → вынесен в SharedComponents
- Удалён мёртвый код: `isPressed` в HabitToggleCard, лишний `enumerated()` в CheckInView

---

## [0.2.4] — 2 марта 2026

### Added
- **Подтверждение удаления** — свайп привычки показывает confirmation dialog перед удалением
- **Undo/Redo** — стрелки ↩ ↪ в хедере привычек: отмена и возврат последних 5 изменений (добавление, редактирование, удаление, перемещение)

### Fixed
- **Корректный расчёт аналитики** — `dayStatus` и все вычисления используют привычки, существовавшие на конкретную дату, а не текущий набор
- **"Perfect" дни** — считаются только дни с реальными 100% (все привычки выполнены), а не >75%
- **Недельные метки** — в помесячной аналитике недели обрезаются по границам месяца
- **toggleCheckin записывает все привычки** — при тогле значения пишутся для всех активных привычек, что исключает ложные 100%
- **Стабильная сортировка** — привычки с одинаковым процентом больше не меняются местами при каждом рендере

### Changed
- Кнопка Edit убрана — привычки всегда в режиме перетаскивания с iOS drag handles
- Тап по привычке открывает редактирование (вместо кнопки-карандаша)

## [0.2.3] — 28 февраля 2026

### Added
- **Growth Language** — все тексты переписаны в мотивирующем тоне: «Пауза» вместо «Не выполнено», «Есть прогресс!» вместо «Частично», «Перерыв» вместо «Пропуск»
- **Push-уведомления** — одно мягкое ежедневное напоминание о чекине в выбранное время. Настройка в Settings > Напоминания

### Changed
- Текст в «О проекте» обновлён в духе Growth Language
- Карточка «Перерыв» в годовой аналитике: оранжевый цвет вместо красного (нейтральнее)

## [0.2.2] — 28 февраля 2026

### Added
- **Per-habit стрики на чекине** — рядом с каждой привычкой отображается "🔥 N дней", если серия >= 2 дней
- **Per-habit хитмапы (Year)** — в годовой аналитике отдельный GitHub-style мини-хитмап для каждой привычки со стриком и счётчиком отметок
- **Сброс вкладки Progress** — повторный тап по иконке "Progress" возвращает к текущему месяцу из любого экрана

### Changed
- Навигация по дням: с карточки "Сегодня" теперь можно перелистнуть на "Завтра"
- Будущие дни дальше завтрашнего заблокированы для перехода из всех экранов (месяц, неделя, год)

## [0.2.1] — 27 февраля 2026

### Added
- **Подробная аналитика (Year)** — отдельный экран: completion rate, стрики, рейтинг привычек, помесячная разбивка
- **Подробная аналитика (Month)** — аналогичный экран для месяца: completion rate, стрики, рейтинг привычек, понедельная разбивка
- **Навигация по дням** — стрелки ◀ ▶ в DayProgressView для переключения между днями
- **Карточка "День года"** — счётчик текущего дня (58/365), пропущенных дней для прошлых годов, всего дней для будущих
- **GitHub-style heatmap** — 5-уровневая градация цвета (none/low/medium/high/full) на основе процента выполнения
- **Хаптик-фидбек** — тактильная отдача на все интерактивные элементы вкладки "Прогресс"

### Changed
- Completion rate теперь считает по привычкам (8 из 10 отметок = 80%), а не по дням (2/2 = 100%)
- Недельные лейблы в аналитике: единый формат "2 – 8 Feb" / "26 Jan – 1 Feb"
- Подсказки в карточках Year view укорочены для корректного отображения на русском
- "Дней" → "Дни" в подсказках карточек

### Removed
- Фильтрация по привычке (чипы "Все / Sleep / Nutrition / ...") — убрана как неиспользуемая

## [0.2.0] — 27 февраля 2026

### Added
- **Чекин на лету** — привычки сохраняются мгновенно при нажатии, без кнопки "Готово"
- **Чекин за сегодня и вчера** — sliding-переключатель дня (🌙 Вчера / ☀️ Сегодня) с плавной анимацией
- **Переключение языка на лету** — в настройках: Системный / Русский / English без перезапуска
- **Автотема (системная)** — приложение следует за системной темой iOS (авто / светлая / тёмная)
- **Поздравление при 100% чекине** — полноэкранный блюр + крупный текст + конфетти + хаптик
- **Стрики в поздравлении** — при серии от 2 дней показывает "🔥 N дней подряд!"
- **Тёмная иконка (iOS 18+)** — автоматическая смена иконки приложения при тёмной теме устройства

### Changed
- Все строки локализации — динамические computed properties для смены языка в рантайме
- Описание в "О проекте" обновлено — убрано упоминание "только вчерашний день"
- Сегодняшний день в прогрессе теперь отображает реальный статус чекина (заливка + оранжевая рамка)
- Оранжевый контур для "сегодня" — единый стиль во всех экранах прогресса (месяц, неделя, год, день)
- Week total и Year summary теперь учитывают сегодняшний день в подсчётах
- Тема применяется через UIKit `overrideUserInterfaceStyle` — мгновенное переключение, включая sheets

### Fixed
- Сегодняшний чекин не отображался в прогрессе (месяц, неделя, год, день показывали пустой контур)
- Стрик считался неверно при чекине вчерашнего дня (не учитывал что сегодня тоже выполнено)
- Повторный быстрый тогл последней привычки вызывал багованное поздравление (таймеры не отменялись)
- Тапы проходили сквозь экран поздравления к элементам под ним
- Переключение на системную тему не работало в Settings sheet (SwiftUI `preferredColorScheme(nil)` не обновлял sheet)

### Removed
- Кнопка "Готово ✓" на экране чекина
- Экран празднования "День записан!" (заменён на полноэкранное поздравление с блюром)
- Кнопка "Изменить" после сохранения (больше не нужна — можно тогглить свободно)

## [0.1.0] — 20 февраля 2026

Первый нативный MVP.

### Added
- Трекинг привычек с ежедневным чекином
- Прогресс: год / месяц / неделя / день с тепловой картой
- Управление привычками (до 10): создание, редактирование, удаление, drag & drop
- Тёмная тема с анимированной сценой день/ночь
- Локализация ru/en (автоопределение по системе)
- Серии (стрики) — лучшая и текущая за месяц
- Конфетти при сохранении чекина
- Хаптик-фидбек на тогл и сохранение
